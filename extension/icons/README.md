# Bull Icons

Inspired by classic charging bull anatomy: **lowered head, curved horns, shoulder hump, forward charge**.

| File | Description |
|------|-------------|
| `bull-charge` | Side-profile charge — **default** |
| `bull-head-on` | Head-on, horns spread, front legs raised |
| `bull-silhouette` | Clean side silhouette |
| `bull-horns` | 3/4 head focus, horns hero |
| `bull-charge-dark` | Filled side charge on black |
| `bull-bold` | Bold expressionist shapes |

```bash
python3 scripts/build-icons.py bull-head-on
python3 scripts/build-icons.py --all
```
